# Gold Business AI Chatbot 🤖✨

This is an AI-powered chatbot specifically designed and developed to automate customer support for a gold and jewelry business. It helps customers get instant answers to their queries and learn more about the business.

---

### 🚀 Key Features
* **200+ QA Dataset:** Capable of handling over 200 business-specific customer questions accurately.
* **Owner & Business Details:** Provides trusted information about the shop's history, credibility, and the owner (my father).
* **Gold Rate & Purity Guide:** Smart assistance to guide users about gold purity (22K, 24K) and general pricing queries.
* **24/7 Availability:** Instant automated customer service without any delay.

---

### 🛠️ Tech Stack (To Be Used)

* **Frontend/UI:** Streamlit (For a clean and interactive web interface)
* **AI/LLM Integration:** Google Gemini API / OpenAI API / LangChain
